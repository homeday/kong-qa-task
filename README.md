# mocha-test-repo

A minimal Node.js repository configured with Mocha and Chai for testing.

Quick start (bash):

```bash
cd mocha-test-repo
npm install
npm test
# get report
npm test:report
```

Files:
- `test/headers.test.js` — test the header info
- `test/add.test.js` — functional tests
- `test/items.perf.test.js` — performance tests
- `test/kong.js` — environment setting

Report:
one sample report is in `reports_sample` folder
